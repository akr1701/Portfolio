A sleek, dark-themed personal portfolio website built with pure **HTML, CSS & JavaScript** — no frameworks, no build tools, just clean code.

---

## ✨ Features

- **Custom animated cursor** with smooth trailing effect
- **Scroll progress bar** at the top of the page
- **Typing animation** cycling through developer roles
- **Scroll reveal animations** — sections fade in as you scroll
- **Animated skill bars** with percentage indicators
- **Animated stat counters** (years, roles, technologies, certifications)
- **Dark cyberpunk aesthetic** with cyan, purple & green accent colors
- **CSS grid layout** — fully responsive
- **Back to top button**
- **Zero dependencies** — runs directly in any browser

## 🌐 Deployment

You can host this portfolio for free on any of these platforms:

**GitHub Pages**
1. Push code to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your site is live at `https://akr1701.github.io/Portfolio/`

**Netlify / Vercel**
1. Drag and drop the `index.html` file onto [netlify.com/drop](https://app.netlify.com/drop)
2. Get an instant live URL
